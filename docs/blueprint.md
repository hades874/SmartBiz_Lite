# **App Name**: SmartBiz Lite

## Core Features:

- Multi-format Data Upload: Accept Excel, CSV files, Google Sheets URLs, and manual data entry with Bangla support.
- AI-Powered Sales Predictions: Provide 7-day, 30-day, and 90-day sales forecasts using the Gemini API.
- Smart Inventory Recommendations: Generate low stock alerts and reorder recommendations, powered by the Gemini API tool.
- Payment & Cash Flow Tracking: Track current cash position and project 30-day cash flow, identifying patterns in payment data using the Gemini API tool.
- Customer Segmentation: Segment customers (high-value, regular, at-risk, lost) using RFM analysis with the Gemini API tool.
- Automated Reports: Generate weekly/monthly reports with key metrics, action items, and recommendations, leveraging the Gemini API tool.
- Dashboard: Display key metrics, quick actions, visual charts, and alerts in a single-page dashboard.

## Style Guidelines:

- Primary color: Forest green (#22C55E) to evoke trust, growth, and stability.
- Background color: Light pastel green (#E6F7EA), a desaturated hue of the primary, for a calm backdrop.
- Accent color: Emerald green (#059669), analogous to the primary, to draw attention to key interactive elements.
- Body and headline font: 'Inter' (sans-serif) for a clean, modern look.
- Use clean, modern icons from Lucide React to represent different data types and actions.
- Implement a card-based layout for easy scanning and access to different insight categories.
- Use subtle transitions and animations to provide feedback on user interactions (e.g., data uploads, report generation).
